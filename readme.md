# 窗口功能
+ 窗口1：图像显示 + 障碍物感知结果（2d/3d） + 车道线显示（可配置等级显示）
+ 窗口2：3d显示 + 车道线显示 + 自车模型 + 障碍物模型（3d框） + 导航轨迹（驾驶状态颜色）（取融合结果）
+ 窗口3：状态+帧率显示 自车驾驶开启功能 + 地图 + 定位 + 传感器
+ 窗口4：重要消息打印  can通信 + 定位 + odom
+ 窗口5：车速、加速度、选择的障碍物距离）
